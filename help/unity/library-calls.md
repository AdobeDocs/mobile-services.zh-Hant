# 呼叫資料庫{#making-calls-to-the-library}

當您想從指令碼呼叫外掛程式時，請匯入命名空間：

* **C#:** 使用 `com.adobe.mobile;`

匯入命名空間後，您可以透過ADBMobile類別的靜態方法直接對外掛程式進行呼叫。
