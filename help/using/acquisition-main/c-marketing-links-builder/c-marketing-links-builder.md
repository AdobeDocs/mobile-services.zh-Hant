# 行銷連結建立器{#marketing-links-builder}

{#eol}

建立器可協助您建立行銷連結，將使用者路由至應用程式商店及應用程式或網頁連結目的地。行銷人員可以根據規則選擇目的地，例如使用者裝置的裝置類型或作業系統。

可追蹤並報告的行銷連結，讓行銷人員可將下載、應用程式重新啟動及其他應用程式內事件歸因於這些連結。
