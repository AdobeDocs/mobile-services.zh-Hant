# 行銷連結建立器{#marketing-links-builder}

建立器可協助您建立行銷連結，以便將使用者引導至應用程式商店和應用程式或網頁連結目的地。行銷人員可以根據規則選擇目的地，例如裝置類型或使用者裝置的作業系統。

可以追蹤和報告行銷連結，讓行銷人員可以將下載、應用程式重新整理及其他應用程式內事件歸因於這些連結。
