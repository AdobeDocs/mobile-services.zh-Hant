# 「管理應用程式設定」{#manage-app-settings}

管理應用程式包含設定應用程式和管理多個變數和量度。

>[!IMPORTANT]
>
>如需編輯此頁面，您必須為行動應用程式管理員群組的成員。若要確定您是否此為群組的成員，請按一下&#x200B;**[!UICONTROL 管理工具]** &gt; **[!UICONTROL 使用者管理]** &gt; **[!UICONTROL 群組]** &gt; **[!UICONTROL 行動應用程式管理員]**。
